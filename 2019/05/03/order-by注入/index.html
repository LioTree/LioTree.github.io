1.order by语法

ORDER BY{column_name[ASC|DESC]},[,…n]

1.1 asc和desc后面还可以加其他列进行多次排序

  • 后面的列同样可以用于注入

1.2 order by后的运算

1.2.1 数字运算失效

  • 即order by后出现数字运算时,无论运算结果是多少,order by总会按照第一列排序
