burpsuite和ss同时使用

首先，在浏览器那里设置burpsuite的代理，然后打开burpsuite，在User options的Connections里可以找到Upstream Proxy Servers和SOCK5 Proxy，两者都是用来设置代理链的

网上已有的文章都说直接在Upstream Proxy Servers或者SOCK5 Proxy设置ss的代理就行了，这大概率是行不通的，因为Upstream Proxy Servers其实设置的是http代理，而ss客户端默认的是sock5代理，如果要使用Upstream Proxy Servers的话ss要改用http代理。burpsuite的SOCK5 proxy要勾选Do DNS lookup over SOCKS proxy才能正常使用

linux下推荐使用Upstream Proxy Servers，因为linux上的ss-qt5没有pac功能，而Upstream Proxy Servers可以在Destination host那里设定使用代理的域名，而SOCK5 proxy只能所有流量都走ss代理
