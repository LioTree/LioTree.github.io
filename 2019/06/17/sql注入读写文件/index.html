开启outfile和load_file

mysql中有一个叫做secure_file_priv的选项限制了写文件和读文件

  • secure_file_priv=null
    不允许写文件或读文件
  • secure_file_priv=路径
    只能在该路径下写文件或读文件
  • secure_file_priv=
    随便写随便读

可以用show global variables like '%secure%';或者select @@global.secure_file_priv;
查看secure_file_priv

linux在/etc/mysql/my.cnf中的[mysqld]下面设置secure_file_priv
windows在mysql目录的my.ini中的[mysqld]下面设置secure_file_priv
