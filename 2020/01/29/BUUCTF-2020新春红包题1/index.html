BUUCTF_2020新春红包题1

  1. 预期解
  2. 非预期解

两个利用点：

  • class B的serialize方法可以执行代码
  • class B的set方法可以写入文件

题目的限制:

  • class A的cleanContents和getForStorage限制了最后写入的文件内容是数组的形式。

但这不影响php的解析

  • class B的getCacheKey方法限制了文件路径包含uniqid()的结果。

可以想到去爆破文件名或者结合WarmUp中的技巧去跨目录

  • class B的getCacheKey方法还限制写入的文件后缀名不能为php。

可以想到写入一个其他后缀的文件，再写入一个.user.ini,利用auto_prepend_file去包含第一个文件。之所以选用.user.ini而不是.htaccess是因为.user.ini容错性更好

  • class B的set方法会在写入的文件前加上exit()。

可以参考p神的文章谈一谈php://filter的妙用，使用php伪协议绕过
